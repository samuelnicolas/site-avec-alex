La balise Body

L'élément Body représente le contenu principal du document HTML. Il ne peux y avoir qu'un élément Body par document,comme pour tous les autres éléments, on peut utiliser les attributs universels

Attributs Désignation Unités acceptées
CSS color Attibue la couleur du texte quand les liens hypertextes sont sélectionnés. code couleur héxadécimale:background-color:#E6E6FA
background L'attribut HTML background de la balise HTML BODY permet d'afficher une image en arrière plan. chemin de l'image
bgcolor L'attribut HTML bgcolor de la balise HTML BODY, permet d'afficher une couleur en arrière plan. code couleur héxadécimale:background-color:#E6E6FA
bottommargin L'attribut HTML bottommargin de la balise HTML BODY, gère la largeur de la marge basse. valeur numérique (par défaut 15)
bgproperties L'attribut HTML bgproperties de la balise HTML BODY, gère le déplacement de l'arrière plan de votre document empty string,fixed
leftmargin La marge à appliquer à gauche du corps de la page. pixel (par défaut 10)
Link La couleur du texte à utiliser pour les liens non visités. couleur
