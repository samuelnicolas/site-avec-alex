La balise Meta

Il existe plusieurs types de balises < meta > elles fournissent des métadonnées sur la page courante. Ces informations ne sont pas visibles, mais sont traitées par les navigateurs et par les moteurs de recherche.
Les meta tags se placent toujours dans la partie < head > du document. Cette balise donne des informations sur la page HTML, La balise meta et plus précisément la balise meta description permet d’ajouter une description d'une page affichée, ainsi que des mots-clés pour lesquels on souhaite optimiser une page, La balise description doit être placée après la balise < title > et avant la balise de fin d’en-tête comme ceci: < html> < head> < title>Titre de la page < meta name="description" content="description de la page" /> < /head> < /html>

Il existe plusieurs types de balises meta mais nous allons citer ici les plus importantes et les plus utilisées:

  • meta DESCRIPTION
  • meta KEYWORDS
  • meta GENERATOR
  • meta ROBOTS
  • meta AUTHOR
  • meta COPYRIGHT
  • meta VIEWPORT

Assurez-vous que la description de votre page est en adéquation avec le titre mais également le contenu de votre page. Si les moteurs de recherche détectent une incompatibilité entre ces éléments, ils peuvent vous considérer comme des spammeurs. Veillez à ce que la description de votre page soit:

  • un déroulement du titre de la page,
  • un résumé du contenu textuel de la page.
  • Ne dépassez pas 160 caractères, espace compris, pour être sûr que la description s’affiche sans être tronquée dans la page de résultat des moteurs de recherche
  • Faites en sorte d’utiliser des phrases avec des verbes afin que la description ait un sens et pour éviter de passer pour du spam.
  • La description ne doit pas être destinée exclusivement aux moteurs de recherche mais également aux internautes qui viendront sur votre site seulement si la description les séduit et parle de ce qu’ils recherchent.

Les méta-tags sont très communément utilisés pour:

  • fournir une description de la page : < meta name="description" content=" ... " />
  • associer un ensemble de mots clés à la page : < meta name="keywords" content=" ... " />
  • préciser le nom de l'auteur du document : < meta name="author" content=" ... " />


Principaux attributs Désignation Unités acceptées
Description c'est une description de la page texte
Keywords Informe les moteurs de recherche sur le contenu texte
Robots Indique aux moteurs d'indexer ou non votre site web texte
Author Précise l'auteur de la page texte
